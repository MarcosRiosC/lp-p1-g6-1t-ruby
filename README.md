## Ejecutando pruebas ⚙️
_Para validar las palabras dentro de una expresión (suma, resta, etc) se sugiere no dejar espacios_

Esta es la forma correcta
```
var1=var2+var3
```
```
var1 = var2+var3
```

De esta forma no validará las palabras que intervienen
```
var1 = var2 + var3
```
